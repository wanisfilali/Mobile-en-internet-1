Opdracht les 6 : Repository bespreken
Medestudent: Wanis
De repository waar ik vandaag over ga hebben is van een medestudent, Wanis genaamd. Zijn repository is wel gestructureerd, maar niet volledig. Alle opdrachten die de docent ons tijdens de les gegeven had, zijn niet gemaakt. Het eerste bestand dat hij geüpload heeft, was tijdens de negende week van het schooljaar. Hij heeft een map mobile en internet in zijn repository waarin hij bestanden gezet heeft. De opdarcht die hij gemaakt heeft, is enkel van les 1 waar we ons zelf moesten voorstellen. Ik heb de html van deze html bestanden gevalideerd. Volgens de validator stonden er paar fouten in. Hij heeft enkel twee keer commits gedaan voor de rest is er niks volledig. Les 1 is de enige les die hij op zijn repository heeft geplaatst verder heeft hij niks anders geüpload. Het verslag over de website en de website voor de klant staan ook niet in de repository. Daaruit kunnen we besluiten dat er geen nieuwe update op de repository geplaatst is en dat hij dus niet regelmatig bezig is met zijn opdrachten.
